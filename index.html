Direkt från officiella Dexcom API

Dexcom V3 stoder G6 och G7. EU och Sverige bor kopplas via api.dexcom.eu, medan sandbox använder sandbox-api.dexcom.com.

Viktigt for riktig lansering

Dexcom vill att tokens lagras serverside. Frontend-only ar bra for prototypen, men riktig inloggning bor ga via backend eller desktop-wrapper.
